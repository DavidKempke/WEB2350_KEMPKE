What is the difference between TCP/IP and HTTP?

TCP/IP the Protocol for all networking. Where as HTTP is the Protocol for the application layer of the TCP/IP model.

Define the term: URL

Uniform Resource Locator- is an address to a file or resourse on a network and how to retrieve the resourse.

What is the Client/Server Model?

An Application that divides the resourse of the network and serves either equally or unequally among the users on the network.

What two things does the browser do?
  1. Access the Internet
  2. View webpages
